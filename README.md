# Valor Theme

Local Shopify theme workspace for Valor.

## Source

Initial source: `valor-v1.3.1.zip`.

## Store

Development store: `valor-demo-store`.

Use Shopify CLI authentication. Do not commit theme access tokens or passwords.

## Commands

```powershell
shopify theme check
shopify theme dev --environment dev
shopify theme push --environment staging --unpublished
shopify theme package
```

Never push or publish to the live theme without explicit approval.
